🧪 Sistema Unificado de Estándares y Control de Calidad

Registro trazable + módulo para ingreso/revisión de datos por analistas

Flujo recomendado: 1) En Registro/Trazabilidad se administra el estándar, lote, COA, vencimiento y baja. 2) En Control de Calidad el analista usa el código/lote vigente para registrar o revisar el control. 3) Si un material vence o se da de baja, no debe usarse para nuevos controles.
